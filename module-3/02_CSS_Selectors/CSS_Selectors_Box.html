CSS Selectors

CSS can be included

  • inline on any element with a style attribute
  • within the document under a <style> element
  • through an external document reference with a <link href=> element

CSS Rules have a selector and a declaration block

  • universal (*)
  • element (li, a, p, h2)
  • id
  • class
  • descendant combinator (descendant space, child (>), adjacent sibling(+), any sibling(~)
  • pseudo-class (a:hover)

All HTML elements are considered rectangular boxes. The term Box Model is used when talking about design and layout

The box consists of

  • Content
  • Padding - the space between the content and the border. inside an element
  • Border
  • Margin - the space outside of an element. Used to space it from other elements

The box-sizing attributes allows you to specify if the height and width should include the padding and border. border-box => yes, content-box => no. content-box is the default.

The Display Attribute specifies if and how an element should be displayed

A block-level elemement always starts on a new line and takes up the full width available.

Examples: div, h1-h6, p, form, header, footer, section

An inline elemement does NOT start on a new line and takes up only as much width as necessary

Examples: span, a, img

An inline-block is similar to inline but allows you to set a width and height

You make an element inline-block by setting the display attribute
