HTML Intro

< tag attribute="value" >element contents</tag>

  • Tag - element name (a, img, h1)
  • Atribute - HTML modifiers (href, src, alt)
  • Value = value applied to an attribute (http://techelevator.com, spongebob.png, some text)
  • Element - everything within the HTML tag - attributes, values and content (< img src="avatar.png" >)

Semantic HTML clearly defines its contents (header, section, etc)

Forms are useful HTML elements for interacting with a server

A form method can be GET or POST and the action attribute is where the form should send its data to, like an API endpoint
